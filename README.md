## Instalação

### 1. Clone o repositório

```bash
git clone https://github.com/matheushenriquecsb/order-menu-vuejs
```

### 2. Acesse o diretório

```bash
cd order-menu-vuejs
``` 

### 3. Instalando dependências

```bash
npm install
```

### 4. Rode a aplicação

```bash
npm run dev
```

### 5. Acesse a aplicação

```bash
http://localhost:5173
```
